# 🤖 LYRONEX

> Asistente virtual local desarrollado con Python.

Liron es un proyecto personal inspirado en asistentes virtuales como Jarvis.  
Actualmente puede escuchar comandos por voz, responder y ejecutar acciones básicas en la computadora.

---

# ✨ Características

- 🎤 Reconocimiento de voz
- 🧠 Conversión de voz a texto con Whisper
- 🔊 Respuestas por voz
- 💻 Automatización básica
- 🎵 Apertura de Spotify
- 🔁 Escucha continua
- 📴 Comando de apagado

---

# 🛠️ Tecnologías utilizadas

- Python
- pyttsx3
- faster-whisper
- sounddevice
- scipy
